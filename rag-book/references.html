Key Resources and Tools

Benchmarks and Evaluation

  • BEIR: Benchmarking Information Retrieval
  • MTEB: Massive Text Embedding Benchmark
  • FreshStack: Modern RAG evaluation benchmark
  • BRIGHT: Reasoning-intensive retrieval benchmark
  • FollowIR: Instruction-following evaluation dataset
  • InstructIR: Persona-based instruction dataset

Models and Libraries

  • ColBERT: Late interaction retrieval model
  • PyLate: Library for late-interaction models
  • Sentence Transformers: Popular embedding library
  • RAGatouille: Easy-to-use ColBERT wrapper
  • ModernBERT: Improved BERT architecture

Vector Databases with Late Interaction Support

  • Vespa
  • Weaviate
  • LanceDB
